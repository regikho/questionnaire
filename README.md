# questionnaire

<h2>Опросник</h2>
  <p>Состоит из 2 экранов:</p> 
  <li>на первом - список вопросов (разная вариация выбора ответа - только один/несколько ответов) и кнопка "Отправить";</li>
  
  <li>на втором - результат (количество верных и неверных ответов).</li>
